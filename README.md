# Superstore-Data-Analysis-Capstone
## Capstone Project | Excel, Python, SQL, Power BI

## Overview

Analyzed Superstore retail data to evaluate sales, profit, customer behavior, and regional performance and built an interactive executive dashboard.

## Objectives

Analyze sales & profit trends

Identify top products, customers & regions

Measure profit margin and delivery performance

## Tools Used

Excel – Data cleaning

Python – EDA & feature engineering

SQL (MySQL) – Data modeling & analysis

Power BI – Dashboard & visualization

## Key Insights

Technology category generates the highest sales

Central region leads in revenue contribution

Consumer segment contributes over 50% of total sales

Certain high-sales products have low profit margins

Faster delivery correlates with higher customer retention
